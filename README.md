# 🎬 CineMatch — A Neural Movie Recommendation System

A full-stack recommender system built on the **MovieLens (small)** dataset, combining:
- **Neural Collaborative Filtering (NCF / NeuMF)** for personalized user recommendations
- **Content-Based Filtering** (TF-IDF + Cosine Similarity) for item-to-item similarity
- An interactive **Streamlit dashboard** to explore both

> Course Project — AI-Pro Intake 46, Recommender Systems

---

## 📂 Project Structure

```
cinematch/
├── Phase1_Data_Preprocessing.ipynb   # EDA, cleaning, encoding, train/val/test split
├── Phase2_NCF_Training.ipynb         # NCF model training & evaluation
├── Phase3_Item_Similarity.ipynb      # TF-IDF + Cosine Similarity for items
├── data/                             # Raw MovieLens CSV files (not committed)
├── processed/                        # Generated by Phase 1
├── models/                           # Generated by Phase 2 & 3

    ├── app.py                        # Streamlit entry point
    ├── utils.py                      # Shared model/data loading logic
    └── pages/
        ├── 1_User_Page.py            # User recommendations
        └── 2_Item_Page.py            # Item similarity explorer
```

## 🚀 Getting Started

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Add the dataset
Download [MovieLens (small)](https://grouplens.org/datasets/movielens/) and place
`ratings.csv`, `movies.csv`, `tags.csv`, `links.csv` into `data/`.

### 3. Run the notebooks in order
1. `Phase1_Data_Preprocessing.ipynb` → generates `processed/`
2. `Phase2_NCF_Training.ipynb` → generates `models/ncf_*`
3. `Phase3_Item_Similarity.ipynb` → generates `models/cosine_sim_matrix.npz` etc.

### 4. Launch the dashboard
```bash
streamlit run dashboard/app.py
```

## 🧠 Models

| Component | Approach |
|---|---|
| **User Recommender** | Neural Matrix Factorization (NCF): embedding-based MF branch + MLP branch combined |
| **Item Similarity** | TF-IDF on genres + tags, compared via Cosine Similarity |

### Results (MovieLens-small, test set)
- **RMSE:** 0.8639
- **MAE:** 0.6743
